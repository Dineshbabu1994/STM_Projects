STM_Projects – Basic Programming using ST Microcontroller

This project series focuses on learning and implementing basic embedded programming concepts using STMicroelectronics (STM32) microcontrollers. It is designed for beginners who want to build a strong foundation in ARM Cortex-M–based development and understand how to interface essential peripherals.

Project Descriptions
1. LED Blinking (GPIO Basics)

Learn how to configure GPIO pins and control digital outputs by creating a simple LED blinking application. This project introduces clock configuration, pin modes, and delay generation.
